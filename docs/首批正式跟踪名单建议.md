# 首批正式跟踪名单建议

## 1. 目的

这份名单用于把蓝图里的“建议样本池”收成一版真正可执行的首批主库。

这一版我按你的要求重做了分类逻辑，目标是：
- 把中国与欧美公司放进同一套分类体系里。
- 分类口径保持一致，尽量做到 `MECE`。
- 同时兼顾你对品牌标杆、AI 品牌、互联网科技公司、服务文化型企业的兴趣。

## 2. 分类口径

这版名单不再按“中国 / 全球”拆分，而是按**首要研究对象类型**分类。

### 分类原则
1. 每个对象只进入一个类别。
2. 分类依据是“你最希望从什么角度研究它”，不是工商注册行业。
3. 如果某个对象理论上可归入多个类别，则按本项目最核心的研究视角放入唯一类别。
